########## RESEARCH MANAGEMENT AND ARCHIVING INFORMATION SYSTEM #########

Technology Used: 
HTML, CSS, BOOTSTRAP 5, JQUERY - Front-End 
PHP - Backend
MySQL - Database

############ How to simulate and run the system #################
- Used XAMPP to act as a server, place the folder wmsurmis inside the xampp htdocs
- open the xampp mysql(mariaDB) import the sql database if0_37772329_rmis.sql 
- open the system navigate to localhost/wmsurmis
